For the E-Commerce API, I focused on building a scalable product catalog system. The key architectural decision was separating read and write operations using CQRS pattern. For reads, I implemented Redis caching with write-through strategy, achieving 80% cache hit rate. For writes, I used PostgreSQL transactions with proper locking to ensure data consistency. The API is deployed on AWS EC2 with Docker, and I set up health check endpoints for load balancer integration.
